## 🌎 Overview | Descripción General

**EN:** This project delivers a full Business Intelligence lifecycle. From raw data auditing in Excel to high-impact visualization in Power BI, including custom UI design.
**ES:** Este proyecto entrega un ciclo de vida completo de BI. Desde la auditoría de datos crudos en Excel hasta la visualización de alto impacto en Power BI, incluyendo el diseño de interfaz personalizado.

---

## 📂 Project Resources | Recursos del Proyecto

**EN:**
* **Dataset:** Original raw data and processed information (ZIP format).
* **Power BI File (.pbix):** The core interactive dashboard with DAX measures.
* **Design Assets:** Custom background template (.pptx) and UI elements (.svg).
* **Executive Report:** A final PDF summary showing the dashboard in action.

**ES:**
* **Dataset:** Datos crudos originales e información procesada (formato ZIP).
* **Archivo Power BI (.pbix):** El dashboard interactivo con medidas DAX.
* **Activos de Diseño:** Template de fondo personalizado (.pptx) y elementos UI (.svg).
* **Reporte Ejecutivo:** Un resumen final en PDF con el dashboard en funcionamiento.

---

## 🛠️ Technical Process | Proceso Técnico

### 1. Data Cleaning & ETL | Limpieza de Datos
**EN:** Performed data auditing in Excel using advanced functions and Power Query to handle nulls and normalize categories.
**ES:** Auditoría de datos en Excel mediante funciones avanzadas y Power Query para manejar nulos y normalizar categorías.

### 2. UI/UX Design | Diseño de Interfaz
**EN:** Designed a professional, "aesthetic" layout in PowerPoint to enhance readability and user engagement.
**ES:** Diseño de un layout profesional y estético en PowerPoint para mejorar la legibilidad y la experiencia del usuario.

### 3. Data Modeling | Modelado
**EN:** Implemented a Star Schema and DAX formulas for accurate KPI tracking.
**ES:** Implementación de un Esquema Estrella y fórmulas DAX para un seguimiento preciso de KPIs.
