## 🌎 Overview | Descripción General

**EN:** This project involved a comprehensive Data Analysis of **Blinkit**'s billing data. The objective was to transform raw retail information into a strategic, interactive dashboard that allows users to navigate and visualize cleaned and valued data for business decision-making.

**ES:** Este proyecto consistió en un análisis de datos integral de la facturación de la empresa **Blinkit**. El objetivo fue transformar información cruda de retail en un dashboard estratégico e interactivo que permite al usuario navegar y visualizar datos limpios y valorizados para la toma de decisiones de negocio.

---

## 🛠️ Tech Stack | Tecnologías Utilizadas

- **Data Cleaning & ETL / Limpieza de Datos:** - **Microsoft Excel:** Advanced functions for initial data auditing. / Funciones avanzadas para auditoría inicial.
  - **Power Query:** Deep cleaning, text normalization, and data type correction. / Limpieza profunda, normalización de texto y corrección de tipos de datos.
- **Data Modeling / Modelado:** Power BI (Star Schema / Esquema Estrella).
- **Calculations / Cálculos:** DAX (Data Analysis Expressions) for custom KPIs. / DAX para métricas y KPIs personalizados.
- **Visualization / Visualización:** Power BI Interactive Dashboard.

---

## 📈 Key Insights | Hallazgos Clave

**EN:**
* **Performance by Outlet:** Identified which store types (Tier 1, 2, or 3) generate the highest revenue.
* **Product Visibility:** Analyzed the correlation between item visibility and total sales.
* **Consumer Preference:** Breakdown of sales based on "Fat Content" and item type.

**ES:**
* **Rendimiento por Sucursal:** Identificación de qué tipos de tienda (Nivel 1, 2 o 3) generan los mayores ingresos.
* **Visibilidad de Productos:** Análisis de la correlación entre la visibilidad del artículo y las ventas totales.
* **Preferencia del Consumidor:** Desglose de ventas basado en el contenido de grasa y tipo de producto.
